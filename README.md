# SingleTask AI

A minimal, emotion‑aware productivity app that helps you focus on **one task at a time**.  
Powered by AI scoring, clean UI, and a distraction‑free workflow.

🌐 **Live Demo:** https://singletask-ai.vercel.app  
🧠 **Backend:** Spring Boot + Groq AI  
🎨 **Frontend:** HTML + CSS + JS  
🚀 **Hosting:** Vercel (frontend) + Render (backend)

## ✨ Features

- 🧠 **AI Pick Next Task** — chooses the best task based on your emotional state  
- 😴 **Emotion‑Aware Scoring** — tired, overwhelmed, motivated  
- 🎯 **Single‑Task Focus Mode** — one task at a time  
- 📝 **AI Parse Tasks** — converts messy text into structured tasks  
- ✔️ **Complete Tasks** — clean, simple workflow  
- 🗂️ **Grouped Pending & Completed lists**  
- 🧹 **Clear Completed Tasks**  
- 🎨 **Beautiful UI** — glassmorphism, animations, gradients  

---

## 🧠 Emotion‑Aware Task Scoring

The backend uses a custom scoring engine:

- **Tired** → low‑energy, short, easy tasks  
- **Overwhelmed** → smallest, safest tasks  
- **Motivated** → high‑priority, meaningful tasks  

This makes the app feel *human* and adaptive.
